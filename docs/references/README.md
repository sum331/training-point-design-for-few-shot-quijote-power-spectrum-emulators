# Z2 Paper References

This directory is the z2 working reference library.

- `references.bib`: complete 57-entry library, copied from the project-level audited library.
- `current_manuscript_references.bib`: 38-entry BibTeX subset approved for the current formal z2 manuscript.
- `current_manuscript_reference_inventory.md`: inventory scoped to the 38-entry current manuscript BibTeX file.
- `manuscript_reference_usage_status.md/json`: use-vs-do-not-use status for all 57 entries.
- `citation_position_status.md/json`: suggested citation position and citation status for all 57 entries.
- `reference_audit_status.md/json`: authenticity audit status for all entries.
- `citation_pipeline_full_57.md`: audit report for the complete z2 reference library.
- `citation_pipeline_current_38.md`: audit report for the current-manuscript 38-entry subset.

Policy: do not delete the 19 lower-relevance entries from the library; keep them marked as `do_not_use_current_manuscript` unless the manuscript scope changes.
