# Third-party and raw-data notes

- Quijote raw simulation products are not redistributed here. The package
  includes only compact coordinates, design sets, processed bias-field products,
  and validation summaries needed for manuscript review.
- The auxiliary KUN/CSST emulator is treated as a third-party fastmock source.
  Its original license and data files remain with the upstream distribution.
- The public z2 code expects external raw data or emulator backends to be
  configured through environment variables or public config files before
  rerunning full expensive experiments.
